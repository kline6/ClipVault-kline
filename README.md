# ClipVault

智能剪切板管理器 — 从 0 到 1 用 Claude Code 协作构建的 Electron 桌面应用，消耗 token 一亿多。

## 功能

- **自动捕获** — 500ms 轮询系统剪切板，文字和图片实时入库
- **一键复制** — 点击卡片即复制回剪切板，支持置顶/标签分类/搜索
- **图片支持** — 截图自动保存，缩略图预览，全屏大图查看
- **自动清理** — 可设 1h~7d 过期自动删除，置顶记录永不过期
- **托盘常驻** — 最小化到系统托盘后台运行，三种关闭行为可选
- **数据导入导出** — JSON 格式一键备份恢复
- **开机自启** — 设置面板一键开启

## 技术栈

| 层 | 技术 |
|---|---|
| 框架 | Electron 33 |
| 前端 | 原生 JS + Lucide Icons |
| 数据库 | sql.js (WASM, 零 native 依赖) |
| 设计 | Kawaii Neon + 星河粒子特效 |
| 测试 | Vitest (4 套件, 60+ 用例) |
| 打包 | @electron/packager (Windows x64) |
